Intersection

∩\cap∩

The intersection of two sets is the set consisting of only the elements that appear in both sets.

  • A∩BA \cap BA∩B denotes the intersection of sets AAA and BBB.
    • ex. G∩D={Alice}G \cap D = \{Alice\}G∩D={Alice} means that "Alice is a part of both set AAA and set BBB".

Two sets whose intersection is the empty set (in other words, have no element in common) are called disjoint.

Example: Yelp

  • A=A =A= The set of Yelp users who like Biz1
  • B=B =B= The set of Yelp users who like Biz2

The set A∩BA \cap BA∩B is the set of users that liked both Biz1 and Biz2
