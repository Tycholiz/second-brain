Mental Tricks

any number that is 1 followed by zero's (e.g. 100, 10,000 etc.) as an exponent can be figured out by multiplying the number of zeros by the exponential value and adding a 1 in front.

  • ex. to figure out 1003100^31003, we take the number of zeros (222) and multiple it by the exponential value (333), which means the result has 6 zeros (1,000,0001,000,0001,000,000)
