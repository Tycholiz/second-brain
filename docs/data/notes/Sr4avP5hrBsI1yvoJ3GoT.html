Set

A set is basically a grouping of unique (ie. no duplicate) things (ie. members): numbers, fruit, other sets, functions. Sets can be finite or infinite. For example, {1, 2, 5, 6} is a finite set. The set of all the natural numbers is an infinite set.

  • ordering is irrelevant
    • ex. 2,4,6{2, 4, 6}2,4,6 and 4,6,4,2{4, 6, 4, 2}4,6,4,2 represent the same set.

Sets correspond to general.lang.types (Private) in programming languages very well. For example:

  • number type can be looked at as a set of all possible numbers
  • string type is a set of all possible strings
  • 'a' | 'b' | 'c' type corresponds to the {'a', 'b', 'c'} set
  • null type is a singleton type with only one element, the null value (which happens to have the same name)
  • undefined type is a singleton type with only one element, the undefined value (which happens to have the same name)
